# Принцип решения
Программа получает на вход число элементов массива, после идет заполнение массива элементами типа string с помощью цикла for.
Далее создается выходной массив а также счетчик для него, после с помощью функции определения длины string и цикла с условием, находятся элементы с длиной меньше 3 и загружаются в выходной массив.
Нумерация входного и выходного массива различные, далее с помощью цикла выводится окончательный массив.
